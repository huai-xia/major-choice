## 按维度快速筛选

### 按兴趣方向

| 你的兴趣 | 推荐专业 |
|---------|---------|
| 喜欢写代码、做产品 | 计算机科学、软件工程 |
| 喜欢数学、研究算法 | 人工智能、数据科学 |
| 喜欢硬件、物理 | 微电子、机器人工程 |
| 喜欢系统设计、攻防博弈 | 网络空间安全 |
| 喜欢软硬结合、做能动的设备 | 自动化、机器人工程 |

### 按数学基础

| 数学水平 | 适合专业 |
|---------|---------|
| 很强（竞赛级别） | 人工智能、数据科学 |
| 较好（高考 130+） | 计算机科学、微电子、自动化、机器人工程 |
| 一般（但逻辑好） | 软件工程、网络空间安全 |

### 按风险偏好

| 风险偏好 | 推荐专业 | 理由 |
|---------|---------|------|
| 追求高天花板 | 人工智能、机器人工程 | 最前沿，但波动也大 |
| 追求稳定高薪 | 计算机科学、软件工程、网络空间安全 | 需求确定，就业面广 |
| 追求长期稳健 | 微电子、自动化 | 国家战略，产业周期长 |
